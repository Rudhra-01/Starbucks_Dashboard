
## 📌 Project Overview

This project is an interactive **Power BI Dashboard** that analyzes Starbucks beverages based on their nutritional values and beverage categories. The dashboard helps users explore calories, caffeine, sugar, beverage distribution, and category-wise insights through interactive visualizations.

---

## 🚀 Dashboard Features

* 📊 Total Beverage Count
* 🍬 Average Sugar Analysis
* 🔥 Average Calories
* ☕ Average Caffeine
* 📈 Average Calories by Beverage Category
* 🌍 Beverage Category Distribution
* ⚡ Average Caffeine by Category
* 🏆 Top 5 Highest Caffeine Beverages
* 🎛 Interactive Filters

  * Protein Range
  * Beverage Preparation Type

## 📈 Key Insights

* Coffee beverages contain the highest average caffeine.
* Smoothies have the highest average calorie count.
* Frappuccino varieties contribute significantly to the beverage distribution.
* Beverage preparation filters allow dynamic analysis of nutritional values.
* Users can compare calories, caffeine, and sugar across different beverage categories.
